status - проверить статус

clone - клонировать удаленную директорию

add -A - добавить все файлы для отслеживания

commit -m 'Name' - добавить коммит с именем Name

restore - отмена изменений

push - отправить на удаленный репозиторий


Если вы случайно закоммитили ненужный файл или папку в git-репозиторий и уже сделали push, то чтобы удалить все следы этого файла или папки в том числе и из истории, достаточно выполнить:
git rm -r myfolder -f
git commit -m "myfolder is deleted"
git push
